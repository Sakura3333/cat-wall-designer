# 猫墙 DIY 图纸与效果图

这里放了两张可直接打开的 SVG：

- `cat-wall-blueprint.svg`：施工参考图，含尺寸、安装点、材料清单和安全提示。
- `cat-wall-render.svg`：暖色、可爱风格效果图。

参考墙面尺寸为 2400mm × 1700mm。实际制作前建议根据墙体类型调整锚栓、背板和承重点：混凝土/实心砖可以用膨胀螺丝，空心砖和石膏板墙建议增加整块背板或固定到龙骨。

主要材料建议：

- 18mm 多层板或实木板
- 松木方、法式挂条或铝型材横梁
- 膨胀螺丝、石膏板锚栓、角码、三角托、法兰盘
- 8-10mm 剑麻绳、可替换抓板
- 帆布、绒布、海绵、魔术贴
- 防滑垫、包边条、木蜡油

安全建议：高位平台和猫屋都做防脱固定；每个平台至少 4 个固定点，外挑平台加三角托；所有边角倒圆打磨。
